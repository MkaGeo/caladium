Курс лекций "Геологическое моделирование"

Систематизированное изучение процесса построения цифровых геологических моделей

Основные сведения

Цель: Дать систематизированное понимание процесса построения цифровых геологических моделей.

Аудитория: Студенты старших курсов, магистранты, аспиранты, молодые специалисты.

Продолжительность: 14 лекций (7 недель) + практические занятия.

Типы моделей:

Структурные Литолого-фациальные ФЕС Геомеханические Гидродинамические

Международное ПО:

Petrel RMS SKUA/GOCAD MOVE

Российское ПО:

ГеоМап ГЕОС ГЕОМОД ГЕОМИКС РУСЬ ГЕОРАДАР

Этапы моделирования

1. Подготовка и анализ данных

Задачи: Сбор, контроль качества и интерпретация всех доступных данных.

Входные данные:

  • Геолого-геофизические данные: сейсмика, структурные карты
  • Скважинные данные: каротаж, керн, испытания
  • Данные о породах-коллекторах и флюидах
  • Геологическая концепция

Процессы: Загрузка данных, контроль качества, интеграция, предварительный анализ.

Выходные данные: Проверенный набор данных, уточненная геологическая концепция.

2. Структурный каркас

Задачи: Создание цифрового представления геометрии изучаемого объекта.

Процессы:

  • Интерпретация сейсмических данных
  • Корреляция скважинных данных
  • Построение поверхностей горизонтов и разломов
  • Создание 3D сетки

Выходные данные: 3D структурная модель с поверхностями горизонтов, разломами и стратифицированной сеткой.

3. Литолого-фациальная модель

Задачи: Распределение типов горных пород в объеме модели.

Методы:

  • Детерминированные: картирование по данным скважин/сейсмики
  • Стохастические:
    • Объектное моделирование (русла, бары, рифы)
    • Имитация последовательности индикаторов
    • Многточечная статистика

Выходные данные: 3D модель распределения литофаций/петротипов.

4. Модель ФЕС

Задачи: Распределение пористости, проницаемости, насыщенности.

Процессы:

  • Анализ данных по керну и ГИС
  • Распределение пористости (статистические методы, сейсмические атрибуты)
  • Распределение проницаемости (петрофизические зависимости)
  • Распределение насыщенности (капиллярные кривые, J-функции)
  • Апскейлинг для ГДМ

Выходные данные: 3D модели ФЕС, апскейленные свойства.

5. Анализ неопределенностей

Задачи: Оценка достоверности и диапазона неопределенностей модели.

Процессы:

  • Анализ чувствительности
  • Анализ неопределенностей (множественные реализации)
  • Верификация (сравнение с исходными данными)
  • Валидация (соответствие внешним данным)
  • Выбор "оптимальной" реализации

Выходные данные: Оценка неопределенностей, отчет о верификации.

6. Итерации и обновление

Задачи: Включение новых данных и уточнение модели.

Процессы:

  • Анализ результатов верификации/ГДМ
  • Сбор новых данных (бурение, сейсморазведка)
  • Возврат к предыдущим этапам для пересмотра
  • Построение обновленной версии

Выходные данные: Актуальная, уточненная геологическая модель.

7. Использование модели

Применение:

  • Подсчет запасов
  • Построение гидродинамических моделей
  • Планирование геологоразведочных работ
  • Анализ рисков
  • Визуализация геологического строения

Ключевые вызовы

Данные

  • Неполнота данных
  • Противоречивость источников
  • Различное разрешение

Моделирование

  • Переход от точек к 3D распределению
  • Выбор адекватных алгоритмов
  • Интеграция разнородных данных

Верификация

  • Субъективность экспертной оценки
  • Вычислительные затраты
  • Управление неопределенностями
